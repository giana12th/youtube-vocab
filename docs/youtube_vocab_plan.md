# YouTube英語字幕 単語学習システム 方針まとめ

## 背景・目的

- 文法はある程度わかるが、語彙力不足でYouTube英語動画が理解できない
- 特に**抽象的な形容詞・副詞**、次に**動詞**の知識が弱い
- 自分専用のシンプルなツールとして作る

---

## 使い方のイメージ

1. YouTubeで動画を見る
2. 見終わったらURLをCLIに渡す
3. 字幕取得 → 単語抽出 → 説明生成（1〜2分）
4. ターミナル上で4択クイズ（5分以内、10〜15問）
5. 間違えた単語をCSV出力

---

## 作る範囲

### ✅ 作る

- 字幕取得（`youtube-transcript-api`）
- 単語抽出・レベルフィルタ（spaCy + wordfreq）
  - 形容詞・副詞・動詞を優先
  - 高頻出語（上位3,000語相当）を除外（wordfreqの使用頻度スコアで判定）
- 単語の文脈説明生成（Claude Codeのスキルベースで処理）
- 4択クイズ（ターミナル上）
  - 出題形式：英単語 + 字幕の例文 → 日本語の意味を選ぶ
  - 不正解選択肢もClaude Codeで生成（意味が似ている語を混ぜる）
- 間違えた単語のCSV出力（再利用性を考慮）

### ❌ 作らない

- フロントエンド / GUI
- SRS・復習スケジュール管理（Anki連携含む）
- ユーザー認証・クラウド保存
- 音声・発音機能

---

## クイズの設計

```
━━━━━━━━━━━━━━━━━━━━
Q3 / 12

"The results were quite ambiguous at first."

👉 "ambiguous" の意味は？

  1. 明確な
  2. 曖昧な
  3. 否定的な
  4. 重要な
━━━━━━━━━━━━━━━━━━━━
```

- 問題数：10〜15問（動画が長い場合は出現頻度・品詞でフィルタして絞る）
- 出題優先順位：形容詞・副詞 → 動詞

---

## 出力ファイル

```
output/
├── {date}_{videoID}_quiz_result.csv   # 間違えた単語のみ
└── {date}_{videoID}_wordlist.md       # 全単語リスト（参照用）
```

CSVは再利用性を考慮した設計（将来的に別ツールへの取り込みも可能）。

---

## 技術方針

| 処理 | 担当 |
|---|---|
| 字幕取得 | `youtube-transcript-api` |
| 品詞判定・レベルフィルタ | `spaCy` + `wordfreq`（上位3,000語を除外） |
| 単語の文脈説明・選択肢生成 | `claude -p` |
| クイズUI | ターミナル（CLIのみ） |
| 出力 | CSV / Markdown |
